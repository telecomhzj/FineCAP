# FineCAP: Fine-grained Cyclic Augmentation Prompting with Zoom-in Enhancement for Few-shot Learning

> IJCNN 2026

## Abstract
FineCAP is a Vision-Language Model adaptation framework for few-shot learning. It introduces two core mechanisms: the **Fine-grained Visual Enhancement Mechanism (FVEM)**, which uses hybrid attention to zoom in on salient visual objects, and the **Dynamic Textual Enhancement Mechanism (DTEM)**, which enriches textual class features with object attribute information. These two modules work in a cyclic augmentation manner to achieve adaptive fine-grained vision-language alignment.

## Results
FineCAP achieves state-of-the-art performance on 11 few-shot benchmarks.
